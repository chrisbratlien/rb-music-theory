h1. cb-music-theory

h2. Overview


basic music theory including notes, note intervals, scales, chords

h3. Note

A Note has an Integer value indicating its place on the chromatic scale.  Asking a
note its octave or name will give you an answer compatible with this 
<a href="http://www.harmony-central.com/MIDI/Doc/table2.html">MIDI note number chart</a>.

h3. NoteInterval

A NoteInterval is used to transform a Note into a new Note by adding the value
of the NoteInterval to the value of the original Note.

h3. Scales and Chords

Both of these classes share a superclass RootNoteWithIntervals.  Both require
a root Note and an array of NoteIntervals which may be applied to the root
Note into the various Notes in the Scale or Chord.

caveat: the intervals array should include a NoteInterval of zero to truly
reveal the root Note.
