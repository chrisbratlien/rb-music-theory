h1. cb-music-theory

h2. Overview

Basic twelve-tone music theory including notes, note intervals, scales, chords

h3. Note

A Note has an Integer value indicating its place on the chromatic scale.  Asking a
note its octave or name will give you an answer compatible with this 
<a href="http://www.harmony-central.com/MIDI/Doc/table2.html">MIDI note number chart</a>.

h3. NoteInterval

A NoteInterval is used to transform a Note into a new Note by adding the value
of the NoteInterval to the value of the original Note.

Note + NoteInterval => Note

<pre><code>
  irb(main):025:0> Note.new("C")
  => #<Note:0x2679f4c @value=60>
  irb(main):026:0> Note.new("C").name
  => "C"
  irb(main):027:0> Note.new("C").plus_interval(NoteInterval.new(12))
  => #<Note:0x2674524 @value=72>
  irb(main):028:0> Note.new("C").plus_interval(NoteInterval.octave)
  => #<Note:0x2675438 @value=72>
  irb(main):029:0> Note.new("C").plus_interval(NoteInterval.octave).name
  => "C"
  irb(main):030:0>
</code></pre>


You can chain NoteIntervals to each other to produce a new NoteInterval of their
sum.

NoteInterval + NoteInterval => NoteInterval
<pre><code>
irb(main):046:0> NoteInterval.new(5).plus_interval(NoteInterval.new(7))
=> #<NoteInterval:0x267396c @value=12>
</code></pre>


h3. Scales and Chords

Both of these classes share a superclass RootNoteWithIntervals.  Both require
a root Note and an array of NoteIntervals which, when applied to the root
Note, yield the actual Notes in the Scale or Chord.

* caveat: the intervals Array must contain a NoteInterval of zero to represent
the root Note.

For convenience, the major scale has been included as a class method.  Notice
that it passes an array of intervals containing NoteInterval.unison for the
root note (same as NoteInterval.new(0)).
<pre><code>
  class Scale < RootNoteWithIntervals

    def self.major_scale(root_note)
      Scale.new(root_note,
        [
          NoteInterval.unison,
          NoteInterval.maj2,
          NoteInterval.maj3,
          NoteInterval.per4,
          NoteInterval.per5,
          NoteInterval.maj6,
          NoteInterval.maj7
        ])
    end
  end
</pre></code>
